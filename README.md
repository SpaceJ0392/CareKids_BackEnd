# AI를 활용한 육아 지원 종합 플랫폼

AI를 활용한 육아 지원 종합 플랫폼 CareKids입니다!  
기존의 분산되어 있던 육아 정보를 통합하여 사용자의 편의성과 공공성을 확보하기 위한 목적의 플랫폼입니다. 🙂
